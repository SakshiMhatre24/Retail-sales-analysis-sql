# Retail-sales-analysis-SQL
SQL-based analysis of a retail database to extract insights on customers, orders, products, and sales performance.


## Database Tables
- Customers
- Orders
- Order Items
- Products

## Analysis Includes
- Data validation using row counts
- Filtering by city and sales channel
- Aggregation using GROUP BY
- Percentage calculations using subqueries
- Sorting to identify top cities and product categories

## Tools & Concepts Used
- SQL (DDL & DML)
- Database schema design with primary and foreign keys
- Filtering using WHERE
- Aggregations using GROUP BY
- Subqueries for percentage calculations
- Sorting with ORDER BY
